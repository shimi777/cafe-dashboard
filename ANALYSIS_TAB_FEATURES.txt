📊 המרכיבים העיקריים:

1️⃣ METRICS (KPIs)
   ├─ סה"כ שבועות: מספר השבועות בנתונים
   ├─ שבוע מחזק: איזה שבוע הביא הרבה ביותר הכנסה
   ├─ סה"כ הכנסה: סכום כל ההכנסות בחודש
   └─ ממוצע לשבוע: הכנסה ממוצעת לכל שבוע

2️⃣ STACKED BAR CHART - משקל כל שבוע בעמידה ביעד
   
   הגרף מראה:
   └─ עמודה לכל שבוע
   ├─ גובה העמודה = תרומת השבוע ליעד החודשי (%)
   ├─ צבע:
   │  ├─ 🟢 ירוק: תרומה טובה (מעל 50%)
   │  ├─ 🟡 צהוב: תרומה סבירה (30-50%)
   │  └─ 🔴 אדום: תרומה נמוכה (מתחת 30%)
   ├─ טקסט בראש כל עמודה: 
   │  ├─ אחוז התרומה
   │  └─ מספר השבוע
   └─ קו מפריד אופקי:
      └─ מציין את היעד הממוצע לשבוע (100% ÷ מספר שבועות)

3️⃣ טבלה מפורטת של שבועות
   ├─ שבוע: מספר וסמל
   ├─ הכנסה: סה"כ הכנסה בשבוע
   ├─ עסקאות: מספר עסקאות בשבוע
   ├─ ימים: מספר ימים שיש בהם עסקאות
   ├─ משקל בחודש: איזה אחוז מסה"כ ההכנסה החודשית
   └─ תרומה ליעד חודשי: כמה אחוזים מהיעד השבוע תרם

4️⃣ סטטיסטיקה מתקדמת
   ├─ הכנסה ממוצעת לשבוע
   ├─ סטיית תקן: מדד של תנודתיות
   │  └─ אם גדול = תנודות גדולות בין שבועות
   │  └─ אם קטן = עקביות בתמיד
   └─ השבוע הטוב ביותר: איזה שבוע הביא הכנסה מקסימלית

═══════════════════════════════════════════════════════════════

💡 דוגמה עם הנתונים שלנו:

עם 4 השבועות שלנו וכל שבוע צריך לתרום ~25% ליעד:

📉 Week 1 (85% ביצוע):
   └─ תרומה: ~21% - 🔴 אדום (מתחת ליעד)

📈 Week 2 (115% ביצוע):
   └─ תרומה: ~27% - 🟢 ירוק (מעל ליעד)

📊 Week 3 (95% ביצוע):
   └─ תרומה: ~24% - 🟡 צהוב/ירוק (סביב ליעד)

🚀 Week 4 (125% ביצוע):
   └─ תרומה: ~27% - 🟢 ירוק (טוב מאוד)

🎯 איך להשתמש:

1. בחר את קבצי ה-HTML שלך
2. עבור לטאב "📉 ניתוח"
3. ראה:
   ✅ איזה שבוע הביא הכנסה הכי הרבה
   ✅ מה היא התרומה של כל שבוע ליעד
   ✅ מה התנודתיות בין השבועות
   ✅ שבוע מדויק עם הנתונים המלאים

🔢 הגרף בפרט:

X-Axis: שבוע 1, שבוע 2, שבוע 3, שבוע 4
Y-Axis: % תרומה ליעד חודשי

כל עמודה:
├─ גובה = תרומה בעמודה
├─ צבע = איכות התרומה
├─ טקסט למעלה = אחוז + שם
├─ קו מפריד בגובה 25% (100/4 שבועות)
└─ Hover = פרטים נוספים

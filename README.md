# 📊 דשבורד ניתוח מכירות - בית קפה

דשבורד אינטראקטיבי לניתוח מכירות בית קפה הבנוי עם Streamlit ו-Plotly.

## 🚀 התקנה והפעלה

### שלב 1: התקנת ספריות
```bash
pip install -r requirements.txt
```

### שלב 2: הפעלת הדשבורד
```bash
streamlit run cafe_dashboard.py
```

הדשבורד ייפתח אוטומטית בדפדפן בכתובת: `http://localhost:8501`

## 📁 דרישות הקובץ

הקובץ Excel צריך לכלול את העמודות הבאות:
- **תאור** - שם המוצר
- **כמות** - כמות שנמכרה
- **סכום** - סכום ללא מע"מ
- **סכום כולל מעמ** - סכום כולל מע"מ
- **מחיר למנה** - מחיר ליחידה
- **קטגוריה** - קטגוריית המוצר

## 📊 תכונות הדשבורד

### 1️⃣ מבט על
- KPIs עיקריים (סה"כ הכנסות, ממוצע יומי, סה"כ פריטים)
- התפלגות הכנסות וכמויות לפי קטגוריה
- סטטיסטיקות מפורטות לפי קטגוריה

### 2️⃣ מוצרים מובילים
- 15 המוצרים המובילים בהכנסות
- 15 המוצרים הנמכרים ביותר (כמות)
- זיהוי מוצרים חלשים לשיקול הסרה

### 3️⃣ ניתוח פארטו (80/20)
- גרף פארטו אינטראקטיבי
- סיווג מוצרים לקטגוריות A, B, C
- זיהוי 20% המוצרים שמייצרים 80% מההכנסות

### 4️⃣ המלצות לקידום
- **מטריצת BCG מותאמת:**
  - ⭐ כוכבים - ביקוש גבוה + רווחיות גבוהה
  - 🐄 פרות מזומנים - ביקוש גבוה + רווחיות בינונית/נמוכה
  - ❓ סימני שאלה - ביקוש נמוך + רווחיות גבוהה
  - 🐕 כלבים - ביקוש נמוך + רווחיות נמוכה

- **המלצות ספציפיות:**
  - מוצרים מומלצים לקידום
  - מוצרים לשקול הסרה

### 5️⃣ טבלת נתונים
- צפייה בכל הנתונים עם סינונים
- מיון לפי כל עמודה
- אפשרות להוריד את הנתונים כ-CSV

## 💡 טיפים לשימוש

1. **העלה את קובץ ה-Excel** דרך הסייד-בר משמאל
2. **נווט בין הטאבים** לניתוחים שונים
3. **השתמש בסינונים** בטאב "טבלת נתונים" לחיפוש ממוקד
4. **הורד תוצאות** בפורמט CSV לשימוש נוסף

## 🎯 תובנות עסקיות שהדשבורד מספק

- זיהוי המוצרים הרווחיים ביותר
- מציאת הזדמנויות לקידום
- זיהוי מוצרים שכדאי להסיר מהתפריט
- הבנת התפלגות המכירות לפי קטגוריות
- אופטימיזציה של מלאי ותפריט

## 🎯 איך זה עובד?

### עבור כל יעד:

**1. הזנת נתונים:**
- הקובץ החודשי מכיל את המכירות בפועל
- אתה מגדיר את היעדים בסייד-בר

**2. המערכת מחשבת:**
```
אחוז עמידה = (ביצוע בפועל / יעד) × 100
```

**3. התצוגה:**
- 🟢 **ירוק** אם >= 100% (עומדים ביעד!)
- 🔴 **אדום** אם < 100% (מתחת ליעד)

### דוגמה:
```
יעד מגדלים: 125
מכרנו: 115
אחוז עמידה: 115/125 = 92% 🔴

יעד הכנסות: 110,000 ₪
הכנסנו: 125,000 ₪
אחוז עמידה: 125,000/110,000 = 114% 🟢
```

שלב 1: התקנת Python
────────────────────
אם אין לך Python:
1. כנס ל- https://www.python.org/downloads/
2. הורד את Python 3.8 ומעלה
3. הרץ את ה-installer
4. ⚠️  חשוב: סמן את "Add Python to PATH" בזמן ההתקנה!

בדוק שהתקנת בהצלחה:
   פתח Terminal/Command Prompt וכתוב:
   python --version
   
צפוי לראות משהו כזה:
   Python 3.10.0

שלב 2: התקנת הספריות
───────────────────
1. שמור את כל הקבצים בתיקייה אחת
2. פתח Terminal/Command Prompt בתיקייה זו
   (בחלונות: בעכבר ימני בתיקייה > Open in Terminal)
3. הרץ את הפקודה הזו:
   
   pip install -r requirements.txt
   
   זה יתקין את כל הספריות הדרושות
   
   אם זה לא עובד:
   pip install -r requirements.txt --user

שלב 3: הרצת האפליקציה
──────────────────────
בעדיין באותו Terminal, הרץ:

   streamlit run app.py
   
זה יפתח אוטומטית דפדפן עם הממשק שלך!
(אם לא נפתח, כנס ל: http://localhost:8501)

❓ שאלות נפוצות:
───────────────────────────────────────────────────────────────────────────────

Q: איפה אני מקבל את קובץ ה-HTML?
A: מהמערכת שלך (POS/AccuPOS/אחר דומה)
   אתה אמור להיות בעל אפשרות "דוח" שיוצא HTML

Q: איך אני יודע שהקובץ HTML תקין?
A: פתח אותו בדפדפן (Firefox/Chrome)
   אתה צריך לראות טבלה עם טרנזקציות

Q: מה אם אני מקבל שגיאה?
A: בדוק:
   1. קובץ ה-Python בעברית? (לפעמים זה בעיה)
   2. קודוד UTF-8 בקובץ HTML?
   3. קובץ HTML מ-AccuPOS?

Q: האם אני יכול להפעיל את זה על קבצים מרובים?
A: כן! העלה קובץ אחר, זה הורץ ניתוח חדש

Q: איך אני עוצר את האפליקציה?
A: בטרמינל לחץ Ctrl+C

Q: האם אוכל להפעיל את זה לעד?
A: כן, פשוט הרץ: streamlit run app.py
   כל פעם שתצטרך

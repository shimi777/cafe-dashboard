# ✅ רשימת משימות ליישום הפרויקט

## שלב 1: הכנה והתקנה ⚙️

- [ ] הורד את כל הקבצים למחשב שלך
- [ ] פתח terminal/command prompt
- [ ] נווט לתיקיית הפרויקט: `cd path/to/project`
- [ ] התקן את הספריות הנדרשות: `pip install -r requirements.txt`
- [ ] בדוק שההתקנה עברה בהצלחה: `streamlit --version`

## שלב 2: הכנת הנתונים 📊

בחר באחת מהאפשרויות:

### אפשרות א': יש לך קובץ HTML
- [ ] שים את קובץ ה-HTML בתיקיית הפרויקט
- [ ] הרץ המרה: `python html_to_excel.py קבלות.html נתוני_קפה.xlsx`
- [ ] בדוק שהקובץ נוצר בהצלחה
- [ ] (אופציונלי) הוסף קטגוריות: `python add_categories.py נתוני_קפה.xlsx`

### אפשרות ב': יש לך Excel מוכן
- [ ] וודא שיש עמודות: תאור, כמות, סכום, סכום כולל מעמ, מחיר למנה, קטגוריה
- [ ] אם חסרה עמודת קטגוריה: `python add_categories.py הקובץ_שלך.xlsx`

### אפשרות ג': רוצה לנסות עם דוגמה
- [ ] הרץ: `python create_example_data.py`
- [ ] יווצר קובץ `cafe_data_example.xlsx`

## שלב 3: הפעלת הדשבורד 🚀

## בעיות נפוצות ופתרונות 🔧

### לא מצליח להתקין את הספריות?
```bash
pip install --upgrade pip
pip install -r requirements.txt --user
```

### הדשבורד לא נפתח?
- בדוק שאין תהליך אחר שרץ על פורט 8501
- נסה פורט אחר: `streamlit run cafe_dashboard.py --server.port 8502`

### הטקסט העברי מקולקל?
- וודא שהקובץ נשמר ב-UTF-8
- השתמש ב-`encoding='utf-8-sig'` בשמירה

### הגרפים לא מוצגים?
- רענן את הדפדפן (Ctrl+F5)
- נקה cache: לחץ על ה-hamburger menu → Clear cache

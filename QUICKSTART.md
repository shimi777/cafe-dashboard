# 🚀 מדריך התחלה מהירה

## צעדים ראשונים

### 1. התקנה (פעם אחת)
```bash
pip install -r requirements.txt
```

### 2. המרת HTML ל-Excel (אם צריך)
אם יש לך קובץ HTML מקבלות:
```bash
python html_to_excel.py קובץ.html נתוני_בית_קפה.xlsx
```

או הרץ והקלד את הנתיבים:
```bash
python html_to_excel.py
```

### 3. הפעלת הדשבורד
```bash
streamlit run cafe_dashboard.py
```

הדשבורד ייפתח בדפדפן בכתובת: http://localhost:8501

---

## 🧪 רוצה לנסות עם נתוני דוגמה?

הרץ:
```bash
python create_example_data.py
```

זה יצור קובץ `cafe_data_example.xlsx` שאפשר להעלות לדשבורד.

---

## 📊 מה הדשבורד כולל?

1. **מבט על** - סטטיסטיקות כלליות
2. **מוצרים מובילים** - הכי מצליחים והכי חלשים
3. **ניתוח פארטו** - כלל 80/20
4. **המלצות לקידום** - מה לקדם ומה להסיר
5. **טבלת נתונים** - כל המידע עם סינונים

---

## ❓ שאלות נפוצות

**ש: מה עושים אם יש שגיאה בהתקנה?**
ת: נסה:
```bash
pip install --upgrade pip
pip install -r requirements.txt --upgrade
```

**ש: איך עוצרים את הדשבורד?**
ת: לחץ Ctrl+C בטרמינל

**ש: איך פותחים את הדשבורד שוב?**
ת: פשוט הרץ שוב:
```bash
streamlit run cafe_dashboard.py
```
